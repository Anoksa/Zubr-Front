О чемпионате

  • Цель проведения

    Создание условий для популяризации сферы инноваций и высоких технологий среди подрастающего поколения, поддержка и вовлечение талантливых детей в научно-техническое творчество

  • Задачи

    1. Поиск и разработка инновационных технологий и методов работы с детьми.
    2. Создавать интеллектуальную образовательную среду, мотивирующую каждого учащегося повсеместно использовать свои знания и умения с целью изменения окружающей обстановки по принципам высокотехнологичного производства.
    3. Способствовать разработке и реализации детских инновационных проектов с помощью высокотехнологичного оборудования и программного обеспечения, создающих условия для формирования современных и будущих профессиональных компетенций, навыков инженерного мышления.
    4. Создавать условия для организации опытно-экспериментальных работ по освоению новых технологий, новых видов профессиональной деятельности, форм профессионального обучения.
